# <a name="recommended-adapters"></a>推奨されるアダプター
> [!NOTE]
> Windows Mixed Reality PC アダプターに関する推奨事項を修正するために、を更新します。 最新のものを定期的に確認してください。

[Windows Mixed Reality 互換 PC](windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md)には、イマーシブヘッドセットの接続に必要な完全な HDMI および USB 3.0 ポートが搭載されていない可能性があります。 その場合は、ヘッドセットを接続するアダプターが必要になります。 アダプターによっては、PC やヘッドセットケーブルに接続するためのカプラーの追加が必要になる場合もあります。 ここでは、必要になる可能性があるアダプターの種類の一覧と、特定のアダプターモデルに関するいくつかの推奨事項について説明します。

## <a name="windows-mixed-reality-compatible-usb-bluetooth-adapters"></a>Windows Mixed Reality 互換 USB Bluetooth アダプター

2018の後にヘッドセットがリリースされた場合、アダプターは不要で、コントローラーはヘッドセットとペアになります。

2017以前に出荷されたデバイスには、コントローラー用のアダプターが必要です。 PC に [Windows Mixed Reality モーションコントローラー](controllers-in-wmr.md)のペアリング用の Bluetooth が搭載されていない場合は、Plugable USB Bluetooth 4.0 低エネルギーマイクロアダプターのようなアダプターを使用できます。

> [!NOTE]

## <a name="using-external-usb-30-hubs-with-windows-mixed-reality-headsets"></a>Windows Mixed Reality ヘッドセットでの外部 USB 3.0 ハブの使用

PC に十分なオープンな USB ポートがない場合は、Windows Mixed Reality ヘッドセットを外部 USB 3.0 ハブで使用できます。 ヘッドセットに十分な電力を供給するためには、外部 USB 3.0 ハブが **AC アダプターを介して外部から給電されている必要があり** ます。

外部 USB 3.0 ハブを使用する場合は、次のガイダンスにも注意してください。 
* USB ハブは USB 3.0 ハブである必要があります。
* USB 3.0 ハブは、PC の USB 3.0 ポートに接続されている必要があります。  
* USB 3.0 ハブは、外部電源になっている必要があります (AC アダプターに電力が供給されている必要があります)。
* 48 w 電源アダプターが搭載された Plugable 10 ポートの USB 3.0 SuperSpeed Hub は、Windows Mixed Reality ヘッドセットとほとんどの Pc で適切に動作することがわかっています。
