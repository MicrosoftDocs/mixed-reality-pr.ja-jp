UE 4.26 を使用している場合は、AR セッションを開始した後で QR コードの追跡を初期化する必要があるため、次のブループリントの設定を使用して短い遅延を追加することをお勧めします。

![遅延を含む ARCapture 切り替え関数のブループリント](../images/qr-codes-img-01.png)

# <a name="425"></a>[4.25](#tab/425)
