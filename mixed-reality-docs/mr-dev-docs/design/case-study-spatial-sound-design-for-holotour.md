## <a name="the-tech"></a><span data-ttu-id="8ae7a-107">技術</span><span class="sxs-lookup"><span data-stu-id="8ae7a-107">The tech</span></span>

<span data-ttu-id="8ae7a-110">空間サウンドは、ユーザーがオンにする方向を示すための手掛かりを提供します。または、スペース内に表示するホログラムがもっとあることをユーザーに知らせます。</span><span class="sxs-lookup"><span data-stu-id="8ae7a-110">Spatial sound provides cues to indicate a direction the user should turn or to let the user know there are more holograms to see within their space.</span></span> <span data-ttu-id="8ae7a-111">また、サウンドをホログラムに直接接続し、ホログラムがユーザーからの方向と距離を継続的に更新することもできます。</span><span class="sxs-lookup"><span data-stu-id="8ae7a-111">We can also attach a sound directly to a hologram and continually update the direction and distance that the hologram is from the user.</span></span> <span data-ttu-id="8ae7a-112">この手法を使用すると、そのオブジェクトから直接サウンドが送られているように見えます。</span><span class="sxs-lookup"><span data-stu-id="8ae7a-112">This technique makes it seem as if the sound is coming directly from that object.</span></span>
