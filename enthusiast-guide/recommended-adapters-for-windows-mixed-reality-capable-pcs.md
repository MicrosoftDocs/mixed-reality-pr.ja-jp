# <a name="recommended-adapters"></a>推奨されるアダプター
> [!NOTE]
> Windows Mixed Reality PC アダプターに関する推奨事項を修正するために、を更新します。 最新のものを定期的に確認してください。

[Windows Mixed Reality 互換 PC](windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md)には、イマーシブヘッドセットの接続に必要な完全な HDMI および USB 3.0 ポートが搭載されていない可能性があります。 その場合は、アダプターが必要です。 一部のアダプターでは、PC またはヘッドセットケーブルに接続するためのカプラーも追加する必要があります。 使用可能なアダプターの種類と、特定のモデルについていくつかの推奨事項を紹介しました。

## <a name="windows-mixed-reality-compatible-usb-bluetooth-adapters"></a>Windows Mixed Reality 互換 USB Bluetooth アダプター

2018の後にヘッドセットがリリースされた場合、アダプターとコントローラーをヘッドセットにペアリングする必要はありません。

2017以前に出荷されたデバイスには、コントローラー用のアダプターが必要です。 PC に [Windows Mixed Reality](controllers-in-wmr.md#pair-motion-controllers)のペアリング用 bluetooth を搭載していない場合は、プラグ可能な USB Bluetooth 4.0 低エネルギーマイクロアダプタなどのアダプタを使用できます。

> [!NOTE]
> Microsoft では、どのアダプターの機能、信頼性、または品質の製造、テスト、認定、または保証も行いません。 任意のアダプターの質問、懸念事項、または問題については、アダプターの販売者または製造元にお問い合わせください。

## <a name="using-external-usb-30-hubs-with-windows-mixed-reality-headsets"></a>Windows Mixed Reality ヘッドセットでの外部 USB 3.0 ハブの使用

PC に十分なオープン USB ポートがない場合は、外部 USB 3.0 ハブで Windows Mixed Reality ヘッドセットを使用できます。 外部 USB 3.0 ハブは、ヘッドセットに十分な電力を供給するために、AC アダプターを介して外部から **給電されている必要があり** ます。

外部 USB 3.0 ハブを使用する場合は、次のガイダンスにも注意してください。 
* USB ハブは USB 3.0 ハブである必要があります。
* USB 3.0 ハブは、PC の USB 3.0 ポートに接続されている必要があります。  
* USB 3.0 ハブは、外部電源になっている必要があります (AC アダプターに電力が供給されている必要があります)。
* 48 W 電源アダプターを搭載した、プラグ可能な10ポート USB 3.0 SuperSpeed Hub は、Windows Mixed Reality ヘッドセットとほとんどの Pc で適切に動作することがわかっています。
